Software De Facturacion.

Funcionalidad:

El software de facturación proporciona las siguientes funcionalidades:

Login - Registro: Permite a los usuarios registrarse e iniciar sesión en el sistema. Presenta dos botones distintos para cada acción, mostrando una ventana correspondiente según la opción seleccionada.

Ingresar Factura: Ofrece una ventana para ingresar nueva información de factura, con campos correspondientes y botones para agregar imágenes, eliminar imágenes adjuntas y guardar la factura en la base de datos.

Buscar Factura: Presenta una ventana con una tabla que permite buscar facturas por nombre, número o fecha. Además, incluye botones para generar un reporte en Excel con la información de la tabla.


Tecnologías utilizadas:
  • El software está desarrollado utilizando la tecnología C #.

Tareas realizadas:
    Las tareas realizadas en el desarrollo del software incluyen:
  • Implementación del sistema de registro e inicio de sesión.
  • Diseño e implementación de la interfaz de usuario para ingresar nueva factura.
  • Desarrollo de la funcionalidad para buscar y filtrar facturas en la base de datos.
  • Integración de la capacidad para generar reportes en Excel.

Launcher - Updater

Funcionalidad:

El software Launcher - Updater proporciona una solución integral para la gestión de versiones y actualizaciones de juegos. Permite a los usuarios descargar e instalar la última versión del juego de manera eficiente. Además, ofrece la funcionalidad de lanzamiento del juego una vez que se ha verificado que los archivos están actualizados.


Tecnologías utilizadas:
  • Desarrollado en C#.

Tareas realizadas:
  • Desarrollo de un Launcher - Updater capaz de verificar la versión actual del juego.
  • Implementación de la funcionalidad de descarga e instalación automática de actualizaciones desde un servidor remoto.
  • Creación de una interfaz de usuario intuitiva que permite a los usuarios iniciar el juego con un solo clic una vez que las actualizaciones se han completado satisfactoriamente.
